# 问题反馈

## 问题反馈说明

关于网站有关问题的反馈，包括但不限于以下几点：

- 网站技术问题，如网页内容渲染异常、网页引用外部源码异常、网站网络连接异常、浏览器适配问题、网站及网页功能异常等。
- 网站优化问题，如网页渲染优化、页面布局优化、网页结构优化、网页源码优化、链接效果优化、动态效果优化、视觉体验优化等。
- 网站交互问题，如建议更好更反馈/投稿方式等。
- 网站侵权问题，如著作权、商标权等。

## 问题反馈规范

关于网站有关问题的反馈，应当遵循以下规范：

- 反馈行为及内容应当符合[社会主义核心价值观](https://baike.baidu.com/item/%E7%A4%BE%E4%BC%9A%E4%B8%BB%E4%B9%89%E6%A0%B8%E5%BF%83%E4%BB%B7%E5%80%BC%E8%A7%82/3271832)。
- 反馈内容及语言表达应当积极向上与文明礼貌，遵循我国法律法规、我国社会公德与传统美德、国际公约（优先级依次递减）。
- 反馈内容及语言表达不应当涉及违法及敏感问题，如暴力、淫秽、恐怖/极端思想、涉及政治问题等。
- 反馈行为及内容应当合理：
  - 对于反馈行为，用户应当认真对待反馈行为，保证反馈内容无误后提交反馈，从而使得**在反馈得到回复前不再次提交反馈**，共同维护网站秩序。
  - 对于反馈内容，用户应当遵循上述要求，确保反馈内容合理，并且不应当将文章投稿、公益宣传、广告招商的**内容**进行问题反馈（注意仅针对内容，若栏目本身存在问题仍应当进行正常反馈）。
